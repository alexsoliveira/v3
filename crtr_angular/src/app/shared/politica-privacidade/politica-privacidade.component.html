Política de Privacidade

Alguns recursos são necessários para fornecer a funcionalidade principal do site, e sem eles, o site não funcionará corretamente, e por isso, eles são criados por padrão.

Atualmente, utilizamos o recurso de Local Storage do seu navegador, com as seguintes chaves:


Chave Recurso Descrição
_p Local Storage Chave utilizada para a confirmação do seu aceite dos termos e política de privacidade do sistema Tabelionet.
_t Local Storage Chave utilizada para armazenar o Token de Segurança para a comunicação segura entre as aplicações do sistema do Tabelionet.
_u Local Storage Chave utilizada para armazenar os seus dados de identificação e autorizações no sistema do Tabelionet.
_s Local Storage Chave utilizada para armazenar temporariamente o status de uma solicitação selecionada.

As chaves "_t", "_u" e "_s" são alimentadas mediante o seu acesso na sistema do Tabelionet, logo, ao efetuar o acesso, você automaticamente concorda com o uso dessa tecnologia.
